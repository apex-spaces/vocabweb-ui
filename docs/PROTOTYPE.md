## 1. 首页/Landing Page

### 页面布局

**桌面端布局**：
```
┌─────────────────────────────────────────────────────┐
│ Header: Logo | 功能 | 定价 | 登录 | [开始使用]    │
├─────────────────────────────────────────────────────┤
│                                                     │
│              Hero Section (全屏)                    │
│         主标题 + 副标题 + CTA按钮                    │
│              演示动画/截图                           │
│                                                     │
├─────────────────────────────────────────────────────┤
│              核心功能介绍（3列）                     │
│         [划词收集] [智能复习] [数据统计]             │
├─────────────────────────────────────────────────────┤
│              工作流程展示（4步骤）                   │
│         1→2→3→4 带可视化流程图                      │
├─────────────────────────────────────────────────────┤
│              产品截图展示（轮播/网格）               │
│         Dashboard | 复习页 | 统计页                 │
├─────────────────────────────────────────────────────┤
│              用户评价/数据展示                       │
│         "已帮助10万+用户记住100万+单词"              │
├─────────────────────────────────────────────────────┤
│              最终CTA区域                            │
│         "立即开始你的单词之旅" [注册按钮]            │
├─────────────────────────────────────────────────────┤
│ Footer: 关于 | 帮助 | 隐私 | 联系 | 社交媒体        │
└─────────────────────────────────────────────────────┘
```

### 核心组件列表

#### 1.1 Header 导航栏
- **Logo**：左侧，点击返回首页
- **导航菜单**：功能介绍、使用场景、定价、帮助文档
- **登录按钮**：文字链接样式
- **开始使用按钮**：主色调CTA按钮，跳转注册页

#### 1.2 Hero Section（英雄区）
- **主标题**："让每个单词都不再遗忘"（大字号，粗体）
- **副标题**："划词即收集，科学复习，轻松掌握"（中等字号）
- **CTA按钮组**：
  - 主按钮："免费开始" → 注册页
  - 次按钮："观看演示" → 播放产品演示视频（模态框）
- **产品演示**：
  - 动画展示：浏览器插件划词 → 单词卡片飞入单词本 → 复习提醒
  - 或产品截图轮播

#### 1.3 核心功能介绍（3列卡片）
每个卡片包含：
- **图标**：视觉化功能（如：鼠标划词、大脑、图表）
- **功能标题**：
  1. "一键划词收集" 
  2. "科学记忆复习"
  3. "可视化进度"
- **功能描述**：2-3句话说明
- **微交互**：鼠标悬停时卡片轻微上浮 + 阴影加深

#### 1.4 工作流程展示
- **4步流程图**：
  1. 浏览网页时遇到生词
  2. 划词自动收集到单词本
  3. 系统根据遗忘曲线安排复习
  4. 查看学习统计，持续进步
- **可视化**：带箭头的流程图 + 配图
- **交互**：点击每一步展开详细说明

#### 1.5 产品截图展示
- **展示方式**：
  - 桌面端：3列网格布局，展示主要页面截图
  - 移动端：横向滚动卡片
- **截图内容**：Dashboard、复习页、统计页、单词详情
- **交互**：点击放大查看

#### 1.6 社会证明区域
- **数据展示**：
  - 用户数量："10万+ 活跃用户"
  - 单词数量："累计收集 100万+ 单词"
  - 复习次数："完成 500万+ 次复习"
- **用户评价**：2-3条精选评价（头像 + 姓名 + 评价内容）

#### 1.7 最终CTA区域
- **背景**：渐变色或品牌色
- **文案**："准备好开始了吗？"
- **CTA按钮**："免费注册" （大号按钮，居中）
- **补充说明**："无需信用卡，30秒即可开始"

### 关键交互说明

1. **滚动动画**：页面滚动时，各区块从下方淡入（Fade in up）
2. **导航栏固定**：滚动后Header固定在顶部，背景变为半透明毛玻璃效果
3. **CTA按钮悬停**：颜色加深 + 轻微放大（scale: 1.05）
4. **视频播放**：点击"观看演示"打开模态框播放视频，点击遮罩层关闭
5. **平滑滚动**：点击导航菜单平滑滚动到对应区域

### 响应式适配要点

**桌面端（≥1200px）**：
- 内容最大宽度 1200px，居中显示
- 功能介绍 3列布局
- 截图展示 3列网格

**平板端（768px - 1199px）**：
- 内容宽度 100%，左右留 padding
- 功能介绍 2列布局（第3个换行）
- 截图展示 2列网格

**手机端（<768px）**：
- Header 导航菜单收起为汉堡菜单
- Hero Section 标题字号缩小
- 功能介绍、截图展示改为单列垂直布局
- CTA按钮宽度 100%
- 流程图改为垂直排列

```
│                  │    ┌──────────────────────┐     │
│  "让单词记忆      │    │  邮箱输入框           │     │

│                  │    │  密码输入框           │     │

│                  │    │  [记住我] [忘记密码?] │     │

│                  │    │  [GitHub登录]        │     │
```

### 核心组件列表

#### 2.1 页面切换Tab
- **登录Tab** / **注册Tab**：点击切换，当前Tab高亮
- **切换动画**：表单内容淡入淡出切换

#### 2.2 登录表单
- **邮箱输入框**：
  - Placeholder: "请输入邮箱"
  - 实时验证：邮箱格式检查
  - 错误提示：红色边框 + 下方错误文字
- **密码输入框**：
  - Placeholder: "请输入密码"
  - 显示/隐藏密码图标（眼睛icon）
- **记住我**：复选框
- **忘记密码**：文字链接，跳转密码重置页
- **登录按钮**：
  - 全宽按钮，主色调
  - 加载状态：显示loading spinner + "登录中..."
  - 禁用状态：表单未填写完整时置灰

#### 2.3 注册表单
- **邮箱输入框**：同登录
- **密码输入框**：
  - 密码强度指示器（弱/中/强）
  - 实时显示：长度、大小写、数字要求
- **确认密码输入框**：
  - 实时验证：与密码是否一致
- **用户协议**：
  - 复选框 + "我已阅读并同意《用户协议》和《隐私政策》"
  - 协议文字可点击，打开模态框显示详情
- **注册按钮**：同登录按钮样式

#### 2.4 第三方登录
- **分隔线**："─── 或 ───"
- **第三方登录按钮**：
  - Google登录：白色按钮 + Google图标
  - GitHub登录：深色按钮 + GitHub图标
  - 按钮样式：带边框，悬停时背景色变化

#### 2.5 切换提示
- **登录页**："还没有账号？[立即注册]"
- **注册页**："已有账号？[立即登录]"
- 点击切换到对应Tab

#### 2.6 左侧装饰区（仅桌面端）
- **品牌Slogan**："让单词记忆变得简单"
- **插画/背景图**：与产品主题相关的视觉元素
- **渐变背景**：品牌色渐变

### 关键交互说明

1. **表单验证**：
   - 失焦时验证单个字段
   - 提交时验证所有字段
   - 错误提示：红色边框 + 下方错误文字 + 抖动动画
2. **密码强度**：
   - 实时计算：弱（红色）、中（黄色）、强（绿色）
   - 进度条显示强度
3. **提交流程**：
   - 点击登录/注册 → 按钮loading状态 → 成功跳转Dashboard / 失败显示错误提示
4. **第三方登录**：
   - 点击 → 打开OAuth授权窗口 → 授权成功后跳转Dashboard
5. **记住我**：勾选后30天内免登录

### 响应式适配要点

**桌面端（≥1024px）**：
- 左右分栏布局（5:7比例）
- 表单区最大宽度 400px，居中

**平板端（768px - 1023px）**：
- 隐藏左侧装饰区
- 表单区居中，宽度 500px

**手机端（<768px）**：
- 表单区宽度 100%，左右留 20px padding
- 第三方登录按钮改为图标按钮（节省空间）
- Logo移至表单上方居中

### 页面布局

**桌面端布局**：
```
┌─────────────────────────────────────────────────────┐
│ Header: Logo | 搜索框 | [+添加单词] | 通知 | 头像   │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│ Sidebar  │  ┌────────────────────────────────────┐ │
│          │  │ 筛选栏                              │ │
│ [主页]   │  │ [全部] [标签] [来源] [掌握度]      │ │
│ 单词列表 │  │ 搜索: [____] 排序: [最新收集▼]    │ │
│ 复习     │  └────────────────────────────────────┘ │
│ 统计     │                                          │
│ 设置     │  ┌────────────────────────────────────┐ │
│          │  │ 单词卡片网格/列表                   │ │
│          │  │ ┌──────────┐ ┌──────────┐          │ │
│          │  │ │ abandon  │ │ abstract │          │ │
│          │  │ │ 放弃     │ │ 抽象的   │          │ │
│          │  │ │ ⭐⭐⭐   │ │ ⭐⭐     │          │ │
│          │  │ └──────────┘ └──────────┘          │ │
│          │  │ ... 更多单词卡片 ...                │ │
│          │  └────────────────────────────────────┘ │
│          │                                          │
│          │  [加载更多] 或 分页器                   │
└──────────┴──────────────────────────────────────────┘
```

### 核心组件列表

#### 4.1 筛选工具栏
- **标签筛选**：下拉多选，显示所有标签
- **来源筛选**：按收集来源（网站、手动添加等）
- **掌握度筛选**：未学习、学习中、已掌握、需复习
- **搜索框**：实时搜索单词或释义
- **排序选项**：
  - 最新收集
  - 最早收集
  - 字母顺序（A-Z）
  - 复习次数
  - 掌握程度

#### 4.2 视图切换
- **网格视图**：卡片式，适合浏览
- **列表视图**：表格式，信息密度高
- **切换按钮**：图标按钮（网格/列表icon）

#### 4.3 单词卡片（网格视图）
每个卡片包含：
- **单词**：大字号，粗体
- **音标**：小字号，灰色
- **主要释义**：1-2条核心释义
- **掌握度指示**：星级（⭐⭐⭐）或进度条
- **标签**：彩色小标签（最多显示2个）
- **操作按钮**（悬停显示）：
  - 播放发音
  - 编辑
  - 删除
  - 添加到复习
- **点击卡片**：跳转单词详情页

#### 4.4 单词列表（列表视图）
表格列：
- **单词** | **音标** | **释义** | **标签** | **掌握度** | **复习次数** | **最后复习** | **操作**
- 每行可展开显示更多信息
- 支持多选：批量操作（删除、添加标签、导出）

#### 4.5 批量操作栏
选中单词后显示：
- **已选中 X 个单词**
- **批量操作**：
  - 添加标签
  - 移除标签
  - 加入复习计划
  - 导出为CSV/Anki
  - 删除

#### 4.6 空状态
没有单词时显示：
- 插画：空书本
- 文案："还没有收集单词"
- CTA按钮："添加第一个单词" 或 "安装浏览器插件"

### 关键交互说明

1. **筛选联动**：
   - 选择筛选条件 → 实时更新列表
   - 筛选条件显示为标签，可点击移除
2. **搜索**：
   - 输入时实时搜索（防抖 300ms）
   - 高亮匹配的文字
3. **排序**：
   - 点击排序选项 → 列表重新排序（带动画）
4. **卡片交互**：
   - 悬停：显示完整操作按钮 + 卡片轻微上浮
   - 点击：跳转详情页
   - 长按（移动端）：进入多选模式
5. **无限滚动/分页**：
   - 桌面端：滚动到底部自动加载更多
   - 移动端：分页器（避免性能问题）

### 响应式适配要点

**桌面端（≥1024px）**：
- 网格视图：4列布局
- 列表视图：完整表格

**平板端（768px - 1023px）**：
- 网格视图：3列布局
- 列表视图：隐藏部分列（复习次数、最后复习）

**手机端（<768px）**：
- 强制网格视图，2列布局
- 筛选栏收起为"筛选"按钮，点击展开抽屉
- 排序改为底部弹出选择器
- 卡片简化显示（只显示单词、释义、掌握度）


## 5. 复习页

### 页面布局

**桌面端布局**：
```

│ Header: Logo | 进度条 (5/23) | [暂停] | [退出]     │
```
